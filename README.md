# Simpel OLS model som kan forudsige prisen på en fodbold spiller
Modellen kan forudsige prisen på en fodbold spiller ud fra spillerens 
- Overall (0-99)
- Overall potientale (0-99)
- Alder
- Position(er)

Lavet som et hygge eksperiment.

Dataen er fra kaggle og er anvender data fra FIFA 22 karriere mode.

## Perfomance
- MAE: ~750.000 eur
- R2: 0.89
